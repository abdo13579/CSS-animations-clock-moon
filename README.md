# Animations Collection

This repo contains simple CSS and HTML animations. Each subfolder demonstrates a different animation effect.

## Contents

- **clock/**
  - `index.html`, `style.css`: A simple animated clock using CSS and HTML.
- **moon/**
  - `index.html`, `style.css`: A simple moon animation using CSS and HTML.

## Usage

1. Open the `index.html` file in any subfolder with your web browser to view the animation.
2. You can modify the `style.css` file to experiment with the animation styles.

---

Feel free to explore and customize these animations for your own projects!
